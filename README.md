# banlist-checker

A command-line tool to check if a Yu-Gi-Oh! card is currently banned in a user-specified format, and returns the banlist status of that card in that format, or, if the format is Genesys, the point cost of that card.

Usage: python3 main.py "card name" "format"

Currently supported formats: TCG, OCG, Goat, and Genesys
